# Discloud CLI

A sample command-line application providing basic argument parsing with an entrypoint in `bin/`.

```sh
dart format .
dart fix --apply
dart fix --apply --code=prefer_const_constructors,prefer_const_constructors_in_immutables,prefer_const_declarations,prefer_const_literals_to_create_immutables
```

```sh
dart compile exe bin/discloud.dart -o discloud.exe
```
